# 큐 (Queue)

- 큐는 데이터를 집어넣을 수 있는 선형(linear) 자료형
- **먼저 집어넣은 데이터가 먼저 나온다.** 이 특징은 FIFO(First In First Out)이라고 부른다.
- 데이터를 집어넣는 것 : Enqueue
- 데이터를 추출하는 것 : Dequeue

: 큐는 순서대로 자료를 처리해야 하는 작업이 존재할 때, 대기열을 구현하거나, 작업을 임시로 저장해두는 버퍼(buffer)로 주로 사용한다.
